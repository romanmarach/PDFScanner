PDFSCANNER - QUICK REFERENCE
=============================

HOW TO RUN
----------
python -m agent <file_path>                      # extract text only (default)
python -m agent <file_path> --mode extract       # same as above
python -m agent <file_path> --mode full          # extract + classify + summarize

Example:
python -m agent data/samples/invoice_test.png --mode full


SUPPORTED FILES
---------------
.pdf, .jpg, .jpeg, .png, .docx


OUTPUT
------
Extracted text saves to: ./output/extracted_text.txt
